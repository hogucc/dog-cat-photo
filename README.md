# dog-cat-photo
dog-cat-photo is a CLI tool for screenshots of dog or cat photo

# Installation
To use dog-cat-photo in your project, run:

`$ npm i dog-cat-photo`

# Usage

`$ dog-cat-photo`

A screenshot of the image of the dog or cat of your choice will be saved.
